Detailed Description

The encoder of the convolutional code of rate 1/2 bits ber symbol as specified in CCSDS 131.0-B-3. The generator polynomials of the encoder are g1(x) = x^6 + x^5 + x^4 + x^3 + 1 & g2(x) = x^6 + x^4 + x^3 + x + 1.

Input: 2048 bit frame Output: Encoded 4096 bit frame

Performs the encoding of the input data. Due to the forecast and set output multiple functions, the input data is a multiple of the output data and the rate of the encoder. The sliding window of the encoder contains 7 (constraintLength) bits and 2 (rate) parity bits are produced, by multiplying the message bits with each of the generator polynomials.
